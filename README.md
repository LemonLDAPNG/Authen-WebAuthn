# Authen::WebAuthn

A library to add [Web Authentication](https://www.w3.org/TR/webauthn-2/) support to server applications

![](image.png){width=100%}

# Status

* Authentication works
* ECC/RSA keys supported
* Registration works with "none" attestation type
